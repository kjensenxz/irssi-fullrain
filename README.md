Installation
------------

Goes into your `~/.irssi/scripts/` folder, `~/.irssi/scripts/autorun/` for 
auto-loading.

Usage
-----
Load the script with `/load fullrain.pl`.

Commands should be clear enough, note that it only uses the active window.

`/rfsay your text`

`/rfsay your text`

`/rftopic your text`

`/rfaway your text`
